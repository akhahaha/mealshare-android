Build
---------------
**BuildConfig**

Please define the following string values in your user `~.gradle/gradle.properties` file:

    # Yelp API keys
    YelpConsumerKey
    YelpConsumerSecret
    YelpToken
    YelpTokenSecret

    # Munch
    MunchFirebaseAddress
